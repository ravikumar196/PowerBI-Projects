## 📌 About This Repository

> Production-ready Power BI dashboards built to solve real business problems — from sales performance to HR retention to executive finance. Each project follows a complete analytics lifecycle: **Raw Data → Power Query ETL → Star Schema → DAX Measures → Executive Dashboard.**
